frontendConfigManager
=====================

这是一个可视化配置项目，通过这个系统可以实现方便的在页面上实现一些复杂的配置。

一共有以下模块

1）消息模块  
2）图片模块  
3）DIALOG模块  
4）对象处理模块  
5）监控模块  
6）图层模块  
7）面板模块  
8）选择器模块  
9）副数据模块  
10）主数据模块  
11）Form模块  
12）API模块
